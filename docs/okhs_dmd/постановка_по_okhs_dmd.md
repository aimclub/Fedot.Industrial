# Анализ научных идей в обсуждении математических основ ML для временных рядов

## Введение

В ходе обсуждения были рассмотрены фундаментальные концепции функционального анализа (теорема представителя,
воспроизводящие ядерные пространства, occupation kernel Hilbert spaces) и их применение в задачах машинного обучения
с временными рядами. Основной фокус — разработка теоретически обоснованного AutoML-фреймворка для классификации,
регрессии и прогнозирования, объединяющего классические ядерные методы с современными подходами на основе дробных
операторов и нейронных сетей.

---

## 1. Теоретические основы

### 1.1 Теорема представителя (Representer Theorem)

*Классическая формулировка*: решение задачи минимизации с регуляризацией в RKHS представимо в виде конечной линейной
комбинации ядерных функций на обучающих данных.  
*Обобщение*: в современных работах теорема распространяется на банаховы пространства (RKBS),
что позволяет использовать L1-регуляризацию и получать разреженные решения.  
*Применение в AutoML*: композитные модели могут быть представлены как линейные комбинации признаковых отображений
различных стратегий (статистических, топологических, частотных), что даёт теоретические гарантии оптимальности.

### 1.2 Воспроизводящие ядерные гильбертовы пространства (RKHS)

*Суть*: пространство функций, где оператор вычисления значения непрерывен, что позволяет использовать ядерный трюк.  
*Связь с глубоким обучением*: современные исследования показывают, что нейронные сети бесконечной ширины эквивалентны
регрессии с нейро-ядерным тангажным ядром (NTK).  
*Применение*: RKHS служит фундаментом для построения occupation kernel Hilbert spaces и методов многомерной регрессии.

### 1.3 Occupation Kernel Hilbert Spaces (OKHS)

*Определение*: гильбертово пространство функций, определённых на целых траекториях (сигналах), а не на точках.  
*Ключевая идея*: каждая траектория представляется occupation kernel'ом, что позволяет интегрировать информацию вдоль
всей траектории.  
*Применение*: идеально для задач прогнозирования, где важна «память» системы (например, дробные дифференциальные
уравнения).  
*Теоретический результат*: OKHS конструируется из RKHS через каноническое отображение;
occupation kernels обеспечивают представление интегральных функционалов.

## 2. Дробные обобщения и динамические системы

### 2.1 Дробные операторы Лиувилля

*Математическая абстракция*: обобщение классического оператора Лиувилля на случай дробного порядка q.
Позволяет моделировать системы с памятью: влияние прошлых состояний затухает по степенному закону.  
*Связь с динамикой*: собственные функции дробного оператора порождают решения через функцию Миттаг-Леффлера (аналог
экспоненты для дробных систем).  
*Применение в прогнозировании*: OKHS с дробными операторами даёт модели, способные учитывать долговременные зависимости.

### 2.2 Функция Миттаг-Леффлера

*Аналогия с экспоненциальным отображением*: как экспонента решает обыкновенные дифференциальные уравнения,
так функция Миттаг-Леффлера решает дробные.  
*Роль в DMD*: в дробном DMD прогноз представляется линейной комбинацией собственных функций,
умноженных на Mittag-Leffler, что естественно для систем с памятью.

### 2.3 Fractional Dynamic Mode Decomposition (DMD)

*Классический DMD*: аппроксимация оператора Копмана для дискретных систем.  
*Дробный DMD*: обобщение на случай дробных производных, позволяющее работать с системами,
не обладающими дискретизацией (например, полиномиальными).  
*Реализация*: через OKHS и occupation kernels строится конечномерная аппроксимация дробного оператора Лиувилля,
что даёт вычислительно эффективный метод.

## 3. Алгоритмические реализации

### 3.1 RKBSCompositeClassifier

*Идея*: использование L1-регуляризации в банаховом пространстве для автоматического выбора значимых ядерных стратегий.  
*PyTorch-реализация*: веса модели обучаются с комбинированным L1/L2-штрафом, что даёт разреженные решения и
интерпретируемость.  
*Ключевые методы*: `fit`, `predict`, `predict_proba`, анализ важности ядер через нормы весов.

### 3.2 OKHSForecaster

*Многомерное прогнозирование*: расширение на произвольный горизонт `forecast_horizon`.  
*Стратегии*:

- **Рекурсивная**: последовательное предсказание одного шага и обновление состояния.
- **Multi-output**: прямое предсказание всех будущих точек через многомерную регрессию.
- **DMD-подход**: с использованием собственных значений и функции Миттаг-Леффлера.
  *Оптимизация*: обучение через PyTorch с возможностью GPU-ускорения и кэшированием весов памяти.

### 3.3 Ядерный бэкенд на PyTorch

*Проблема*: наивные циклы в NumPy медленны.  
*Решение*: все базовые ядра (RBF, Matérn, периодическое, дробное и др.) реализованы с векторизованными операциями в
PyTorch.  
*Оптимизированное вычисление матрицы Грама*: использование pad-тензоров и broadcasting, кэширование весов,
выбор между стандартным и оптимизированным методом в зависимости от размера данных.

### 3.4 Графовые и адаптивные ядра

*SpectralMixtureKernel*: моделирование периодических структур через смесь спектральных компонент.  
*GraphDiffusionKernel*: ядро на основе графового лапласиана для структурированных данных.  
*AdaptiveKernel*: обучаемые параметры (длина масштаба, сигма) через PyTorch-параметры.

## 4. Интеграция в AutoML-фреймворк

### 4.1 UnifiedOKHSAutoML

*Задача*: единая система для классификации, регрессии и прогнозирования временных рядов.  
*Принцип работы*:

- Автоматическое определение типа задачи (по наличию меток и их распределению).
- Для классификации — RKBSCompositeClassifier с ансамблем ядер (разные q и базовые типы).
- Для прогнозирования — OKHSForecaster с оптимизацией дробного порядка q через кросс-валидацию.
- Для регрессии — гибрид с L2-регуляризацией.

### 4.2 OKHSEnhancedAutoML

*Расширения*:

- Анализ памяти временного ряда (автокорреляция, спектральный анализ) для подбора оптимального q.
- Построение доверительных интервалов через бутстрап.
- Скользящее прогнозирование (rolling forecast) с переобучением.

### 4.3 Анализ памяти и выбор q

*Методы*:

- По наклону логарифмической автокорреляции (степенное vs экспоненциальное затухание).
- По доминирующей частоте (низкочастотные → долгая память → большой q).
- Адаптивный q как обучаемый параметр в ядре.

## 5. Экспериментальная валидация

### 5.1 Бенчмарки

*M4*: временные ряды разной частоты (hourly, daily, monthly и т.д.).  
*Monash Forecasting Repository*: разнообразные датасеты (туризм, электричество, трафик).  
*Синтетические данные*: генерация рядов с трендом, сезонностью, сложной динамикой для тестирования.

### 5.2 Модели для сравнения

*Простые*: Naive, SeasonalNaive, LinearRegression, ARIMA, ExponentialSmoothing.  
*Сложные*: DeepAR (планируется), LSTM, а также обёртки для OKHS и RKBS.  
*Наши*: OKHSForecaster с разными q и методами (direct, dmd), RKBSForecaster (адаптированный).

### 5.3 Метрики качества

*Для прогнозирования*: MAE, MSE, sMAPE, MASE.  
*Для классификации*: accuracy, precision, recall, F1, ROC-AUC.  
*Дополнительно*: время обучения, разреженность модели (для RKBS).

### 5.4 Визуализация результатов

*Индивидуальные прогнозы*: графики исходного ряда, прогнозов разных моделей с доверительными интервалами.  
*Сравнение горизонтов*: матрица ошибок для разных h.  
*Анализ разреженности*: важность ядерных стратегий.  
*Сводные таблицы*: усреднённые метрики по всем датасетам.

### 5.5 Процедура эксперимента

1. Загрузка датасета (реального или синтетического).
2. Разделение на train/test с учётом временной структуры.
3. Для каждого горизонта прогноза запуск всех моделей.
4. Сбор метрик и сохранение результатов (CSV, JSON).
5. Генерация отчёта с визуализациями и сводными таблицами.

## 6. Направления дальнейших исследований

*Глубокие RKBS*: интеграция с нейросетями для автоматического построения признаков.  
*Мета-обучение*: перенос оптимальных ядерных стратегий между датасетами.  
*Онлайн-OKHS*: адаптация к нестационарным рядам.  
*Причинный анализ*: расширение OKHS для моделирования причинно-следственных связей.  
*Квантизация и прунинг*: использование разреженности, получаемой в RKBS, для сжатия моделей.

В ходе обсуждения была разработана целостная теоретико-методологическая база для AutoML на временных рядах,

- современные обобщения (RKBS, OKHS);
