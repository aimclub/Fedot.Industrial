# RKHS and OKHS Examples

Status: current forecasting scenario.

The RKHS/OKHS example is a forecasting use case. The public entrypoint is
`forecasting/okhs_forecasting.py`, which builds a current
`benchmark.industrial` suite config with an OKHS model spec and a naive
baseline. Use the shared publication-pack rendering helpers after a real run to
produce aggregate tables and plots.
