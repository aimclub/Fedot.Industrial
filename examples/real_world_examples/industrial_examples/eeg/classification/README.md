# EEG Classification Examples

This folder contains EEG classification notebooks and local-data contexts.

## Files

- `harmful_brain_activity_classification.ipynb`: migrated Kaggle notebook from
  the old outdated examples area.

The notebook expects local EEG competition data in
`examples/utils/data/ts_classification/eeg`. The repository does not commit those
datasets; use
`examples.real_world_examples.current_api.eeg_classification_context` to inspect
the expected paths.
