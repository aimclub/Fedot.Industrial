# Examples Utils

`utils/` keeps shared example infrastructure:

| Path | Purpose |
| --- | --- |
| `current_api/` | Minimal reproducible Industrial entrypoints and JSON manifests used by smoke tests and docs. |
| `data/` | Small tracked fixtures and README-described local data roots, grouped by task type. |

Use `python -m examples.utils.current_api` for the lightweight current API smoke run.
